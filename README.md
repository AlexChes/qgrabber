# Questions Grabber

## How to use

Install `Java` - http://www.oracle.com/technetwork/java/javase/downloads/


Clone repository and run the application.
```
git clone https://github.com/AlexChes/qgrabber
cd qgrabber
bash compile.sh
bash run.sh "PAGE_URL"
```
